усовершенствовал новостное приложение:

В категории есть возможность пользователей подписываться на рассылку новых статей в этой категории.

Если пользователь подписан на какую-либо категорию, то, как только в неё добавляется новая статья, её краткое содержание приходит пользователю на электронную почту, которую он указал при регистрации. В письме есть гиперссылка на саму статью, чтобы он мог по клику перейти и прочитать её.

Если пользователь подписан на какую-либо категорию, то каждую неделю ему приходит на почту список новых статей, появившийся за неделю с гиперссылкой на них, чтобы пользователь мог перейти и прочесть любую из статей.

Добавлено приветственное письмо пользователю при регистрации в приложении.
